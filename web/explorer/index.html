🤖 Telegram Bot — AbsoluteBot

Control your Absolute Blockchain node directly from Telegram.

Setup

  1. Create a bot via @BotFather on Telegram
  2. Copy your BOT_TOKEN
  3. Set environment variable: set TELEGRAM_BOT_TOKEN=your_token
  4. Run: python telegram_super_bot.py
